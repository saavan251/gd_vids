Watching Videos

You can just watch videos without signing in from mobile, tabs, PCs etc. The contents are best viewed with Google Chrome browser. So prefer to use Google Chrome for viewing. Currently few videos are not playing in current version of chrome.Things will get sorted with further updates.

Sharing Videos

You can not only just watch videos but also share them for others to view. To share the video you must follow the steps as described below. The steps are similar as if you had to install a software and then perform certain steps... Plz.. use lan wired connection. Router requires extra setup :P [If you want to use router you will have to perform port forwarding for ports 3000 and 8888 and also make your laptop assign static ip in your router control panel. Only one device can share from one router. Also your ip will be the ip you have assigned to your router...]

  • Initial setup

    1. You must have nodejs intalled in your system. You can visit the website here. You can also follow steps here
    2. After you have installed and checked that node is working fine, you must select a location in you computer where you wish to keep your shared videos. Download the zip from here. Unzip the files and place the folder there.
    3. You must keep in mind not to rename or delete any of the files or folders from this folder. If any such thing happens redownload the setup files and replace/add the modified/deleted files.
    4. Run the install script named install.bat by double clicking it. If no error occurs you are ready to share videos.
    5. enter your dc nick and password in the file named userdata.txt in the form "username,password". example- sinha,pass.
    6. Login to gamedrone website and go to settings and change the ip to your ip (router ip incase you are using your router else the ethernet port ip]. Always perform this step if you change your ip.
    7. Start your server by using the script open_server.bat. You always have to start your server before sharing can occur. This is similar to opening ApexDC or DC++, as long as you keep running the server your videos remain shared. As soon as you turn off your server you go offline. To turn off the server you have to run the script close_server.bat.

  • Adding videos to share

    1. Copy the videos to the folder named share_these_videos.
    2. If your server is already running you do not need to restart it again
    3. Run the script named openserver.bat by double clicking it. This will start your server. (To close the server run closeserver.bat). Your server must be running before you can share a video. To test if your server is running- in browser go to the following ip http://{your_ip}:8888/list.html. This should show server running. If this is not the case then there is some problem in running the server.
    4. Run the script named share_my_videos.bat by double clicking it. This will open a terminal and if no error is found it means that you have successfully shared your videos.
    5. In case if the terminal takes very long time (2-3 minutes or so) to finish. Restart the step 2 again by first closing the terminal. If more videos are shared it will take more time. The time is mainly the hashing time that takes in DC.
    6. Now you have shared your videos and your videos will be stored in the folder my_shared_videos. Donot remove the videos from this folder unless you want them to be removed from your share. To do that follow the steps given in Removing videos from share.

  • Removing videos from share

    1. To remove videos from your shared list .Copy those videos from the folder named my_shared_videos to the folder named remove_from_share.
    2. If your server is already running you do not need to restart it again.
    3. Run the script named openserver.bat by double clicking it. This will start your server. (To close the server run closeserver.bat). Your server must be running before you can share a video. To test if your server is running- in browser go to the following ip http://{your_ip}:8888/list.html. This should show server running. If this is not the case then there is some problem in running the server.
    4. Run the script named remove_my_videos.bat . This will open a terminal and if no error is found it means that you have successfully removed videos from your share.
    5. In case if the terminal takes very long time to finish. Restart the step 2 again by first closing the terminal.
    6. Now you have removed videos from your share. These videos will be stored in the folder unshared_videos.You can place them anywhere form here or delete it if you wish so.

Updating scripts

  1. If in case there arise a need to update the scripts.. you just have to Download the zip from Updates . Unzip the files and replace only the files in the folder where you have shared.
  2. Donot perform any change with the folders.
